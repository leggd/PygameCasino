How to Use
To use the main menu open the main.py file to start playing the game.
Once opened you can choose which game you want to play and select the game.
If you want to exit the main menu press the big red exit button in the top left corner.

Project Description:
This is a Selection screen for users to choose which game they want to play straight from the start of game.

Technologies Used:
Python 3.11
Pygame

Requirements:
To run this game, you will need:

Python 3.9 or higher
Pygame library

Installation Instructions:
Ensure Python 3.9 or higher is installed on your system.

Install Pygame using pip:
pip install pygame

Download the game files to your local machine.

Usage Instructions:
Navigate to the directory containing the game files.

Run the game by executing:
python home.py

Follow the on-screen instructions to play the game. You can place bets, hit, stand, and restart the game after each round.
